성능 미세 최적화 + 예외 버그 보정 묶음
====================================

이번 패치에서 처리한 것
---------------------
1. 채팅 이미지 큐 렌더링을 requestAnimationFrame 기준으로 배치 처리
2. 누락되어 있던 schedulePendingChatImagesRender 복구
3. 이미지 준비 중 / 업로드 중 중복 조작 방지
4. 여러 장 업로드 중 일부 성공 후 실패하면 남은 이미지만 큐에 복구
5. 채팅 업로드 timeout / Cloudinary 설정 오류 메시지 보강
6. 프로필 크롭 업로드 중 중복 클릭 방지
7. 프로필 업로드 timeout / 설정 오류 / 네트워크 오류 문구 보강
8. 프로필 이미지는 업로드됐지만 Firebase 동기화 실패한 경우 재시도 가능한 상태 유지

기대 효과
---------
- 채팅 이미지 큐 UI 리렌더 횟수 감소
- 다중 이미지 업로드 실패 시 중복 전송/중복 복구 감소
- 실사용 중 '버튼 연타', '준비 중 전송', '업로드 타임아웃' 같은 예외 상황 완화
