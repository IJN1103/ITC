## 별도 확인 대상

루트에 `_map-viewport.js`가 남아 있다면 Rules 관련 파일이 아니다.
현재 프로젝트의 실제 개발 분할 파일은 `dev/map-token/_map-viewport.js`다.
단, 루트 `_map-viewport.js`는 현재 `dev/map-token/_map-viewport.js`와 내용이 다르므로 이번 패치에서 자동 삭제하지 않았다.
삭제 여부는 별도 점검 후 결정한다.
