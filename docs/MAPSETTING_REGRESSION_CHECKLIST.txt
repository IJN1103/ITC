맵세팅 최종 회귀 테스트 체크리스트

목적
- 맵세팅 ZIP import 관련 수정 후, 같은 순서로 안정성을 점검하기 위한 공통 체크리스트입니다.
- 배경/전경, imported panel token 생성, 레이어 UI, 속성 매핑, 재적용, 회귀 여부를 한 번에 확인합니다.

기본 원칙
- 가능하면 항상 같은 테스트 ZIP으로 먼저 확인합니다.
- 실패가 나오면 그 단계에서 멈추고 실패 지점을 기록합니다.
- 위치/크기/우선도는 반드시 원래 의도 이미지와 함께 비교합니다.

사전 준비
1. GM 계정으로 실제 세션 방 입장
2. 테스트용 맵세팅 ZIP 준비
3. 원래 의도 이미지 준비
4. 맵세팅 레이어 창이 열리는지 확인
5. 패널 토큰 기본 기능이 이미 정상인지 확인

━━━━━━━━━━━━━━━━━━
A. ZIP 검사 / 적용
━━━━━━━━━━━━━━━━━━
1. 맵세팅 ZIP 업로드
2. 검사 완료 메시지 확인
3. 적용 실행
4. 콘솔 에러 없는지 확인

성공 기준
- 검사 실패 없이 적용까지 진행됨
- 즉시 오류 토스트나 콘솔 에러가 없음

━━━━━━━━━━━━━━━━━━
B. 배경 / 전경
━━━━━━━━━━━━━━━━━━
1. 배경 이미지 적용 확인
2. 전경 이미지 적용 확인
3. 새로고침 후 유지 확인

성공 기준
- background / foreground가 정상 반영됨
- 새로고침 후에도 유지됨

━━━━━━━━━━━━━━━━━━
C. imported panel token 생성
━━━━━━━━━━━━━━━━━━
1. object들이 일반 레이어 DIV가 아니라 패널 토큰으로 생성되는지 확인
2. 생성된 패널 수가 예상 범위와 크게 다르지 않은지 확인
3. 중복 생성 없이 반영되는지 확인

성공 기준
- object가 실제 panel token으로 생성됨
- 의도하지 않은 중복 생성이 없음

━━━━━━━━━━━━━━━━━━
D. 위치 / 크기 / 회전
━━━━━━━━━━━━━━━━━━
1. 원래 의도 이미지와 현재 결과 비교
2. 왼쪽 상단 오브젝트 확인
3. 중앙 큰 오브젝트 확인
4. 오른쪽 영역 오브젝트 확인
5. 회전값이 들어간 오브젝트가 있으면 각도 확인

성공 기준
- 전체 배치가 의도 이미지와 크게 어긋나지 않음
- 특정 오브젝트가 중심/좌상단 기준으로 밀리지 않음
- 크기가 비정상적으로 압축되거나 확대되지 않음

━━━━━━━━━━━━━━━━━━
E. 속성 매핑
━━━━━━━━━━━━━━━━━━
1. memo 반영 확인
2. locked -> 위치 고정 확인
3. coverImageUrl이 있으면 panelBackImage 반영 확인
4. closed가 true면 초기 뒷면 표시 확인
5. clickAction이 있으면 패널 액션 타입/텍스트 반영 확인

성공 기준
- ZIP 속성값이 패널 토큰 필드로 정상 연결됨

━━━━━━━━━━━━━━━━━━
F. 레이어 UI
━━━━━━━━━━━━━━━━━━
1. 맵세팅 레이어 창 열기
2. imported panel 항목이 보이는지 확인
3. 눈 아이콘으로 숨김/표시 확인
4. 순서 변경 확인
5. 새로고침 후 상태 유지 확인

성공 기준
- 레이어 UI와 실제 패널 표시 상태가 일치함
- 순서 변경이 실제 겹침 순서에도 반영됨

━━━━━━━━━━━━━━━━━━
G. 패널 토큰 실동작
━━━━━━━━━━━━━━━━━━
1. imported panel 클릭 액션 확인
2. 더블클릭 전환 확인
3. 위치 고정된 패널 이동 차단 확인
4. 일반 패널 토큰 기능과 충돌 없는지 확인

성공 기준
- imported panel도 일반 패널 토큰 규칙 안에서 안정적으로 동작함

━━━━━━━━━━━━━━━━━━
H. 재적용 / 정리
━━━━━━━━━━━━━━━━━━
1. 같은 ZIP 다시 적용
2. 이전 imported panel 중복 생성 없는지 확인
3. 기존 레이어 목록이 비정상적으로 쌓이지 않는지 확인
4. 새 상태로 정상 교체되는지 확인

성공 기준
- 재적용 시 중복 누적이 없음
- 교체/정리가 정상 동작함

━━━━━━━━━━━━━━━━━━
I. 최종 회귀
━━━━━━━━━━━━━━━━━━
1. 일반 패널 토큰 생성/편집 확인
2. 일반 캐릭터 토큰 이동/편집 확인
3. 채팅 / speak-as / 다이스 기본 확인
4. 맵 줌/팬 후 레이어/패널 상태 확인

성공 기준
- 맵세팅 수정이 다른 핵심 기능을 망가뜨리지 않음
