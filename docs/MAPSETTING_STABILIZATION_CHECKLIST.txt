맵세팅 안정화 최종 회귀 체크리스트

목적
- 맵세팅 ZIP import / 레이어 UI / imported panel token 동기화가 계속 안정적으로 유지되는지 점검
- 큰 기능 추가 전에 최소 한 번은 아래 순서를 끝까지 확인

A. 기본 적용
1. +맵세팅 모달 열기
2. ZIP 선택
3. 적용 완료까지 진행
4. 배경 / 전경 / 오브젝트 레이어가 정상 반영되는지 확인
5. 오브젝트가 일반 레이어 DIV가 아니라 imported panel token으로 생성되는지 확인

B. 배치 / 속성
1. 위치가 ZIP 의도와 크게 어긋나지 않는지 확인
2. 크기가 비정상적으로 커지거나 작아지지 않는지 확인
3. 회전값이 반영되는지 확인
4. memo / locked / cover / closed / clickAction 반영 대상이 정상적으로 이어지는지 확인

C. 레이어 UI
1. 레이어 목록이 background / object / foreground 순으로 정상 표기되는지 확인
2. preview 썸네일이 보이는지 확인
3. 표시/숨김 눈 아이콘이 정상 동작하는지 확인
4. 순서 변경 시 실제 패널 겹침 순서도 같이 바뀌는지 확인

D. 삭제
1. 선택 삭제 시 해당 레이어만 사라지는지 확인
2. 연결된 imported panel token도 같이 삭제되는지 확인
3. 전체 삭제 시 오브젝트 레이어와 연결된 imported panel token이 전부 삭제되는지 확인
4. 전체 삭제 시 배경 이미지도 함께 삭제되는지 확인

E. 재적용 / 새로고침
1. ZIP 재적용 시 이전 맵의 오래된 layerId / visible / order 상태가 남지 않는지 확인
2. 새로고침 후 visibility / order / 삭제 상태가 유지되는지 확인
3. 유령 패널 / 유령 레이어가 다시 나타나지 않는지 확인

F. 위치고정 맵세팅 패널 동작
1. 위치고정 imported panel 위에서 좌클릭 드래그 시 토스트가 뜨지 않는지 확인
2. 위치고정 imported panel 위에서 좌클릭 드래그 시 토큰은 움직이지 않고 맵 팬처럼 자연스럽게 동작하는지 확인
3. 위치고정 imported panel도 필요한 상호작용은 유지되는지 확인
   - 앞/뒷면 이미지가 있는 경우 더블클릭 전환 가능
   - 우클릭 편집창 진입 가능
   - 클릭 액션이 설정되어 있으면 기존 패널 토큰 기준대로 동작

G. 다른 기능 회귀
1. 일반 패널 토큰 클릭 / 더블클릭 / 편집이 기존처럼 되는지 확인
2. 일반 토큰 드래그가 기존처럼 되는지 확인
3. 맵세팅 레이어 UI 수정이 다른 모달/UI를 깨뜨리지 않는지 확인

H. 장면 전환 연동 최종 확인
1. 씬 A에 맵세팅 ZIP을 적용하고 저장한다.
2. 맵세팅 레이어 표시/숨김을 변경한다.
3. 1초 정도 기다린 뒤 씬 B로 전환하고 다시 씬 A로 돌아온다.
4. 표시/숨김 상태가 씬 A에 유지되는지 확인한다.
5. 레이어 순서를 변경한 뒤 씬 B → 씬 A 복귀 시 순서가 유지되는지 확인한다.
6. 단일 레이어 삭제 후 씬 B → 씬 A 복귀 시 삭제 상태가 유지되는지 확인한다.
7. 전체 삭제 후 씬 B → 씬 A 복귀 시 배경/오브젝트/연결 패널 토큰이 삭제된 상태로 유지되는지 확인한다.
8. 새로고침 후 active 씬의 맵세팅 상태가 유지되는지 확인한다.

I. 전체 삭제 버튼 표시 조건
1. 오브젝트와 배경이 모두 있는 상태에서 전체 삭제 버튼이 보이는지 확인한다.
2. 전체 삭제 시 오브젝트 레이어, 연결된 imported panel token, 배경 이미지가 함께 삭제되는지 확인한다.
3. 오브젝트를 개별 삭제해서 배경만 남긴 상태에서도 전체 삭제 버튼이 보이는지 확인한다.
4. 배경만 남은 상태에서 전체 삭제 시 배경 이미지가 삭제되는지 확인한다.

현재 안정화 기준
- 맵세팅 ZIP object는 imported panel token으로 생성되어야 한다.
- imported panel token의 레이어 표시/숨김/순서는 맵세팅 레이어 UI와 동기화되어야 한다.
- 위치고정 imported panel token 위 좌클릭 드래그는 토스트 없이 맵 팬으로 처리되어야 한다.
- 위치고정 imported panel token의 편집창, 앞/뒷면 전환, 클릭 액션은 막지 않는다.
- 전체 삭제는 오브젝트 레이어, 연결된 imported panel token, 배경 이미지를 함께 삭제하는 동작이 맞다.
- active 씬에서 맵세팅 레이어 변경 후 빠르게 씬 전환을 해도 변경 상태가 유지되어야 한다.
