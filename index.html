스탠딩
채팅 시 메시지 박스에 표시되는 이미지를 설정합니다. 예를 들어, "@미소"라는 라벨로 설정 한 경우, 채팅에서 "... @미소" 처럼 더하는 것으로 스탠딩을 변경합니다.
스테이터스
HP나 MP 등 캐릭터에 연동하여 변동하는 스테이터스를 설정합니다. {라벨명}처럼 채팅하면 채팅창에서 현재 값을 불러올 수 있습니다.
매개변수
캐릭터의 거의 변하지 않는 매개변수를 설정합니다. {라벨명}처럼 채팅하면 채팅창에서 값을 불러올 수 있습니다.

🔧 Firebase 연결 가이드
1. console.firebase.google.com 접속
2. 프로젝트 만들기
3. Authentication → 이메일/비밀번호 사용 설정
4. Realtime Database → 만들기 (테스트 모드)
5. 프로젝트 설정 → 웹 앱 추가 → 설정값 복사
6. 이 HTML 파일 상단 <script> 의 firebaseConfig 에 붙여넣기
7. 파일 저장 후 다시 열기

설정 후 모든 방문자가 계정을 만들고 실시간 멀티플레이를 즐길 수 있습니다.
