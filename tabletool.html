🔧 Firebase 연결 가이드
1. console.firebase.google.com 접속
2. 프로젝트 만들기
3. Authentication → 이메일/비밀번호 사용 설정
4. Realtime Database → 만들기 (테스트 모드)
5. 프로젝트 설정 → 웹 앱 추가 → 설정값 복사
6. 이 HTML 파일 상단 <script> 의 firebaseConfig 에 붙여넣기
7. 파일 저장 후 다시 열기

설정 후 모든 방문자가 계정을 만들고 실시간 멀티플레이를 즐길 수 있습니다.
